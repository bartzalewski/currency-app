## ⭐ How To Install?

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

## ✨ What kind of technologies did I use?

- Vue
- VueX
- Axios
- AOS
- SCSS
- ESLint
- Font Awesome
